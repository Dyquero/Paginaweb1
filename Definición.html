El backend en informática es una parte de un software o sitio web que es invisible para los usuarios.

Es lo opuesto al frontend, que se refiere a una interfaz de usuario (IU) de software o sitio web./p>

Mientras que el frontend se conoce como la capa de presentación, el backend sirve como una capa de acceso a datos.
